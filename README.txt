PRISMATIC TRANSIT v15 UPDATE

Replace the existing index.html in the main repository directory with the
index.html in this ZIP. Leave the music folder unchanged.

Changes:
- Glitch decay now runs from 0.10 to 1.00
- Default decay is 0.80
- Old saved decay values are automatically migrated to 0.80
- Star Surf normally keeps decay between 0.60 and 0.97
- The centre of the accumulated trail layer is gently erased each frame
  so it cannot build into a solid white patch
- Music-driven centre glow has been reduced
